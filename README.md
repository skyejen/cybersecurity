I've always been obsessed about technology, how systems talk to each other and the endless whys. Early in 2026 I decided to give cybersecurity a go, and it overconsumed me.

As the time of writing (25/04/2026) I've only been learning for 2+ months, but I've already touched upon so many interesting concepts, it feels like a door into a new dimension opened up. I feels like I lived in a small box and didn't realise it. And now the door is open and the outside is so big it feels like I'm in space.

This repo is to document the concepts, tools and tasks that really got under my skin, impressed me or took a while to solve.
